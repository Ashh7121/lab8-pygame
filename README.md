# lab8-pygame
The program usesthe Python library pygame to open a window and animate 100 squares moving around the screen. Each square is given a random size, colour and starting position, and its speed is calculated so that smaller squares move faster while larger ones move slower. The program runs in a continuous loop where it clears the screen, updates the position of each square using simple x and y (velocity?)values, and redraws them every frame. When a square reaches the edge of the window, its direction is reversed so it appears to bounce off the walls. The animation is updated 60 times per second to keep movement smooth, and the program stops when the user closes the window.
